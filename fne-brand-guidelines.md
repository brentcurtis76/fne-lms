# FNE Brand Guidelines & Design System

## Typography (Tipografías)

### Primary Font: MONT
**Institutional typography** - Base for all communications

**Usage:**
- Titles, highlighted texts, and informational content
- Titles should use uppercase (MAYÚSCULAS)
- **Contrast Rule**: Always use HEAVY and LIGHT versions for contrast

**Available Weights:**
- MONT BLACK / MONT BLACK ITALIC
- MONT HEAVY / MONT HEAVY ITALIC ⭐ **Primary for contrast**
- MONT BOLD / MONT BOLD ITALIC
- MONT SEMIBOLD / MONT SEMIBOLD ITALIC
- MONT REGULAR / MONT REGULAR ITALIC
- MONT BOOK / MONT BOOK ITALIC
- MONT LIGHT / MONT LIGHT ITALIC ⭐ **Primary for contrast**
- MONT EXTRALIGHT / MONT EXTRALIGHT ITALIC
- MONT THIN / MONT THIN ITALIC
- MONT HAIRLINE / MONT HAIRLINE ITALIC

### Secondary Font: EAMES CENTURY MODERN
**Complementary typography** for long texts, paragraphs, or extensive content requiring good readability

**Usage:**
- Long texts and paragraphs
- **For printed base texts: Use BOOK version**

**Available Weights:**
- EAMES CENTURY MODERN BLACK
- EAMES CENTURY MODERN BOLD
- EAMES CENTURY MODERN MEDIUM
- EAMES CENTURY MODERN BOOK ⭐ **For printed texts**
- EAMES CENTURY MODERN BOOK ITALIC
- EAMES CENTURY MODERN REGULAR
- EAMES CENTURY MODERN LIGHT

### Script Font: JUST LOVELY
**Alternative typography** for particular uses - highlights or more informal and cheerful titles

## Gradients (Degradados)

### Gradient Rules
Each logo has a combination of two colors:
- **Primary color** (70% of the band)
- **Darker color** for subtle gradient

**Available Gradient Combinations:**
- NUEVA EDUCACIÓN: Yellow to Brown
- LODGE: Brown to Dark Brown
- SUMMER CAMP: Coral to Red
- RELACIONA: Red to Dark Red
- LOS PELLINES: Green to Teal
- EMPRESA: Teal to Blue

## Logo Usage Guidelines

### Logo Components
- **Isotipo**: Symbol/icon element
- **Logotipo**: Text element

### Protection Area
Equivalent to one instance of the word "NUEVA" on each side

### Minimum Sizes
- **Print**: 15mm minimum
- **Digital**: 42px minimum

### Correct Logo Versions

#### Vertical Versions
- Black/Grayscale
- White on Blue Base background
- Full color
- Color on Blue Base background

#### Horizontal Versions
- Black/Grayscale
- White on Blue Base background
- Full color
- Color on Blue Base background

## Image Usage Guidelines

### Clean Images Requirements
- **High quality and sharp images**
- **Good contrast** for proper logo readability and visualization
- **Colors should align** with the graphic line and color palette
- **Logo placement**: Any corner where reading is most contrasted and legible

### Low Contrast Solution
When adequate contrast or legibility cannot be achieved:
- Use a band with base colors
- **70% opacity**
- **Multiply effect**

## Complementary Graphics

### Patterns (Tramas)
- Generated from linear Isotypes
- Arranged in bands, side by side
- **Purpose**: Add texture to flat backgrounds
- **Application**: Solid color or using gradients
- **Rule**: Always consider color complement

### Decorative Isotopes
- Use linear Isotypes as decorative elements
- **Application**: Subtle appearance in certain places of graphic formats
- **Rule**: Measured application following palette's chromatic line

## Design System for LMS Platform

### Critical Implementation Notes
1. **Primary Typography**: Use MONT HEAVY for headings, MONT LIGHT for body text
2. **Color Priority**: Always combine Family Colors with Base Colors
3. **Blue Base (#00365b)** should be the primary background color for the platform
4. **Nueva Educación Yellow (#fdb933)** as the primary accent color
5. **Gray Base (#e8e5e2)** for secondary backgrounds and cards
6. **Minimum logo size**: 42px for digital applications
7. **Gradient usage**: 70% primary color, 30% secondary color
8. **Image overlays**: Use 70% opacity with multiply effect when needed

### Web-Specific Guidelines
- **Contrast ratios** must meet accessibility standards
- **Responsive logos**: Ensure 42px minimum is maintained across all devices
- **Color consistency**: Use exact HEX values provided
- **Typography hierarchy**: MONT HEAVY for H1-H3, MONT LIGHT for body text
- **Interactive elements**: Use Nueva Educación yellow (#fdb933) for buttons and highlights
- **Background patterns**: Implement subtle tram patterns for visual interest without compromising readability
